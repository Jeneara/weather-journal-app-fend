# Getting Started

To install the application it is necessary to have the Node installed on your machine.

<h2>Installing</h2>
At the root of the application install the dependencies

``` 
npm install
```

Initialize the server

```
node server.js
```
Access the localhost on port 5500

```
http://localhost:5050/
```
